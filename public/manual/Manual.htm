Sistema de tramitação de documentos

 

O sistema de tramitação de documentos tem por função o controle de documentos que são enviados entre os diferentes órgãos da prefeitura de Guarulhos.

            A primeira interface que se apresenta logo que o endereço é digitado na barra de endereços do navegador é a tela de login do usuário, onde é pedido o seu login (nome de usuário) e a senha de acesso ao sistema.

 

 

            Após preenchidos os dois campos clique no botão “Entrar” para ter acesso a interface principal do sistema de tramitações.

           

1 – Menu de Cadastros

           

            Dá acesso aos principais tipos de cadastros do sistema de tramitações tais como: Documentos, Assuntos, Tipos de documento etc.

 

2 – Menu de Consultas

 

            Permite a visualização de diversas consultas em forma de tabelas das informações sobre os documentos.

 

3 – Menu de Impressão

 

            Permite a impressão de consultas geradas no sistema de tramitação de documentos.

 

4 – Menu de Configurações

 

            Dá acesso a personalização do sistema, permitindo que o usuário configure-o de acordo com as necessidades e especificações de cada órgão e setor.

Cadastros      

 

         Permitem o cadastramento das principais informações do sistema que compõem as informações relacionadas aos documentos, o menu “cadastros” é composto pelas opções: “Tipo do documento”, “Assunto”, “Órgão Externo”, “Documento” e “Tramitação” explicados com detalhes mais adiante.
